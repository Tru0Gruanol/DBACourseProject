# DBA Course Project

## 一、项目简介

本项目为数据库课程设计，课题为**托管培训中心信息管理系统**。

#### 项目背景

某托管培训中心需要建立一套信息管理系统，替代原有的人工管理方式，
对学生报名、科目维护、教室排课以及账目收费等日常业务进行统一的数字化管理。

#### 核心功能
1. **学生报名**：查询科目是否满员，满员时提示报名下一期；未满员时选择教师、完成报名登记并生成收费清单
2. **科目管理**：维护培训科目基本信息，支持同一科目由不同等级教师承担，并根据教师知名度设置差异化收费标准与课时报酬
3. **排课管理**：为每个班级安排上课时间、地点，生成学生与教师的上课日程表
4. **账目管理**：记录学生每笔缴费流水，支持开具收据、打印收费清单与催缴欠费查询

#### 系统架构

采用**前后端分离**架构：**前端（Vue.js）** 负责用户界面展示与交互，**后端（Spring Boot）** 负责业务逻辑处理与数据接口，**数据库（MySQL）** 负责持久化存储全部业务数据，前后端通过 **HTTP** 接口进行 **JSON** 格式的数据通信

#### 开发方式

使用 Git 进行版本管理，每日提交代码并记录开发日志，
项目全程托管于 GitHub，地址：https://github.com/Tru0Gruanol/DBACourseProject

## 二、开发环境

| 技术/工具 | 类别 | 在本项目中的作用 |
|-----------|------|----------------|
| Java | 后端语言 | 编写后端业务逻辑，如学生报名、收费计算、满员判断等核心功能 |
| Spring Boot | 后端框架 | 快速搭建后端服务器，提供前端调用的 HTTP 接口 |
| MyBatis | 数据库框架 | 负责后端与 MySQL 之间的数据交互，执行增删改查操作 |
| Maven | 构建工具 | 管理后端项目所依赖的第三方库，自动下载配置 |
| Vue.js | 前端框架 | 构建用户操作界面，包括报名页面、课程列表、账目查询等 |
| Node.js | 前端运行环境 | Vue 项目的运行和构建依赖环境 |
| MySQL | 数据库 | 存储系统全部业务数据，包括学生、教师、科目、班级、账目信息 |
| MySQL Workbench | 数据库工具 | 可视化管理 MySQL 数据库，用于建表、查询和调试 SQL |
| IntelliJ IDEA | 后端 IDE | 编写和调试 Spring Boot 后端代码 |
| VS Code | 前端编辑器 | 编写和调试 Vue 前端代码 |
| Git | 版本控制 | 记录每日代码提交历史，追踪项目开发进度 |
| GitHub | 代码托管 | 远程存储项目代码及开发日志，作为课设提交平台 |
| Postman | 接口调试 | 在前端页面开发前，直接测试后端接口是否正常工作 |

## 四、 基本表的设计
根据**ER**图设计数据库表结构：
**教师表（teachers）** 主键：**teacher_id**
**科目表（subjects）** 主键：**subject_id**
**学生表（students）** 主键：**student_id**
**班级表（classes）** 主键：**class_id** 外键：**subject_id**和**teacher_id**
**学生选课报名表（student_enrollments）** 主键：**enrollment_id** 外键：**student_id**和**class_id**
**账目流水表（accounts）** 主键：**account_id** 外键：**student_id**和**class_id**
