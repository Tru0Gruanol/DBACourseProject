# Role: 项目上下文记忆与会话总结专家

## Background

阶段性任务任务结束，我们需要将本次对话的核心成果、解决的问题以及关键逻辑"持久化"保存到本地，以便在开启新一轮对话时，你（或下一个 Claude 实例）能够通过扫描该文件无缝接轨项目进度。

## Task

请利用你的文件写入工具（`write_to_file`），在**log Claude Code**文件夹中，自动创建并写入一份 Markdown 格式的会话总结文件。

## File Naming Rule

文件命名格式请严格遵循：`Dayxx logcc.md`（例如：`Day01 logcc.md`），确保文件名唯一且按时间排序。

1. **直接执行**：收到此指令后，请直接调用相关工具生成并保存文件，无需向我再次确认或询问结构。
